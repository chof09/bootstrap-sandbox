Breakpoints

While we originally included breakpoints in our CSS variables (e.g., --breakpoint-md), these are not supported in media queries, but they can still be used within rulesets in media queries.

not supported: @media (min-width: var(--breakpoint-sm)) { ... }
supported: @media (min-width: 768px) { .custom-element { color: var(--primary); } }

  • var(--breakpoint-xs); { 0px }
  • var(--breakpoint-sm); { 576px }
  • var(--breakpoint-md); { 768px }
  • var(--breakpoint-lg); { 992px }
  • var(--breakpoint-xl); { 1200px }
